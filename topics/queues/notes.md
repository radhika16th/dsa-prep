## Queues

**Key Idea:** First In First Out (FIFO) data structure.

**Operations:**
- enqueue → add to rear
- dequeue → remove from front
- peek → view front element

**Time Complexity:**
- enqueue: O(1)
- dequeue: O(1)
- peek: O(1)

**Common Use Cases:**
- BFS (Breadth First Search): shortest path
- Level-order traversal
- Simulation (real-world queue behavior)
