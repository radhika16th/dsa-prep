# Trees

## Hierarchical data structure with nodes connected by edges

## Terms
- Root, Parent, Child
- Leaf (no children)
- Height / Depth

## Traversals
- DFS:
  - Inorder (LNR)
  - Preorder (NLR)
  - Postorder (LRN)
- BFS:
  - Level Order

## ⏱️ Time Complexity
Traversal: O(n)

Insert (BST avg): O(log n)

Search (BST avg): O(log n)

## Common Patterns
- DFS (recursive thinking)
- BFS (level-wise processing)
- Divide & Conquer
- Binary Search Tree (BST rules)

## To use when:
- Hierarchical data
- Recursion-based problems
- Search problems (BST)
