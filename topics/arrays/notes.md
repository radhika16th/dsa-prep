## Arrays

### Contiguous Memory + Indexed Access

### Time Complexities
Access: O(1)

Search: O(n)

Insertion/Deletion: O(n)

Append: O(1), (worst case: O(n))

### Common Patterns
- Two Pointers
- Sliding Window
- Prefix Sum

## To use when:
- Need fast index access
- Working with contiguous data
- Space efficient as no memory overhead required for pointers
