## Graphs

## Collection of nodes (vertices) connected by edges

## Types
- Directed Graph (edges have direction)
- Undirected Graph (no direction)
- Weighted Graph (edges have values)
- Unweighted Graph
- Cyclic Graph
- Acyclic Graph (DAG)

### Time Complexities
BFS: O(V + E)

DFS: O(V + E)

Add Vertex: O(1)

Add Edge: O(1)

## Common Patterns
- BFS (Shortest Path in Unweighted Graph)
- DFS (Traversal / Cycle Detection)
- Topological Sort
- Union Find (Connected Components)
- Dijkstra (Shortest Path in Weighted Graph)

## To use when:
- Representing relationships (social networks, maps)
- Finding shortest paths
- Detecting cycles
- Scheduling tasks (dependencies)
- Network routing

## Representations
- Adjacency List (most common)
- Adjacency Matrix
